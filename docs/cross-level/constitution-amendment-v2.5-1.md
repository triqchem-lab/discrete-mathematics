## 一、三进制 Trit 的宪法复位

### 宪法定义

> **三进制 trit 的宪法定义为 {0, 1, 2}**，对应驻波三态：
> - **T₀ = 0**：吸收态 / 基态
> - **T₁ = 1**：平衡态 / 中
> - **T₂ = 2**：表达态 / 阳

这是 **根数学的第一性定义**，其代数结构为 **GF(3) 加法群**，且与 **C3 循环群的表示直接同构**。

**严禁使用 {-1, 0, 1} 表述**：该表述残留电性文明的阴阳二元对立思维，将吸收/表达态简化为正负对立，掩盖了 GF(3) 作为**纯代数循环结构**的本质。

### 两种表示的范畴分离

| 表示 | 身份 | 范畴 | 合法性 |
| :--- | :--- | :--- | :--- |
| **{0, 1, 2}** | trit 的 **本源姿态**，直接对应主权状态机的相位与手性 | 根数学（第一性） | ✅ 宪法唯一合法定义 |
| **代数映射** | 用于拓扑计算时的仿射变换（如差分运算） | 耦合域（工程投影） | ✅ 作为计算方案合法，但禁止与本源混淆 |

### 转换法则（宪法授权）

**拓扑计算时的仿射变换**（仅用于差分/曲率计算）：
\[
\text{Topo}(0) = -1,\quad \text{Topo}(1) = 0,\quad \text{Topo}(2) = 1
\]

**逆映射**：
\[
\text{Enc}(-1) = 0,\quad \text{Enc}(0) = 1,\quad \text{Enc}(1) = 2
\]

此映射在主权 TQ1_0 格式的 `qs[6]` 打包中强制使用（每 5 trit 编码为 0–242 的整数）。

### 为何本源必须是 {0, 1, 2}？

| 宪法要求 | {0, 1, 2} 满足 | {-1, 0, 1} 的缺陷 |
| :--- | :--- | :--- |
| **纯代数循环** | 0→1→2→0 构成完整循环，无正负对立 | 引入人为的"正负"符号，陷入阴阳二元思维 |
| **工程原生** | 直接对应字节编码与硬件 ALU 操作 | 需要额外符号处理层，增加认知负担 |
| **模 3 算术** | 加法/乘法天然封闭在 {0,1,2} | 需频繁处理负数模运算，掩盖代数纯洁性 |
| **与 C3 群表示同构** | 直接对应 ω⁰, ω¹, ω² 的幂次索引 | 需仿射变换才能对齐 |

### 工程实现中的宪法约束

在 `qs[6]` 打包与解包时，必须显式使用转换法则，且 **禁止** 在未转换的情况下将编码值直接用于主权状态机演化。

```c
// 合法代码示例：解包（标准三进制 {0,1,2}）
Trit unpack_trit(uint8_t byte, int pos) {
    int val = (byte / (int)pow(3, pos)) % 3;
    // val 已经是 {0, 1, 2}，直接返回
    return (Trit)val;
}

// 合法代码示例：打包
uint8_t pack_trits(Trit trits[5]) {
    uint8_t byte = 0;
    for (int i = 0; i < 5; i++) {
        // trits[i] 已经是 {0, 1, 2}
        byte += trits[i] * (int)pow(3, i);
    }
    return byte;
}
```

## 二、克里斯托螺线与斐波那契死亡螺旋的范畴分离

### 宪法复位

> **三进制 trit {0, 1, 2} 是根数学的离散格点基底，服务于主权状态机的驻波拓扑演算。它不直接构造任何连续统螺线。克里斯托螺线是神圣几何（√2 正交格点）的离散测地线投影，斐波那契螺旋是死亡几何（φ 二进制递归）的退化投影。三进制的 C3 旋转可合法参与克里斯托离散测地线的角向步进，但禁止将 trit 序列直接等同于螺线生成规则。**

### 螺线类型的范畴分离

| 螺线类型 | 几何基底 | 律算身份 | 与三进制的关系 |
| :--- | :--- | :--- | :--- |
| **克里斯托螺线（神圣几何）** | 正交格点、√2、√3，离散商空间测地线 | 结构学：复三维环面极向/环向缠绕的**离散测地线**在三维的投影 | 三进制 trit 的 C3 旋转（120°）构成其离散角向步进，但螺线本身需正交基（√2）参与 |
| **斐波那契螺旋（死亡几何）** | 二进制递归、黄金比例 φ，无限不闭合 | 密度：光锥矩阵内信息对半丢失的**退化投影** | 三进制与 φ 无关；将三进制序列映射到 φ 螺旋是电性文明的非法代数联想 |

### 宪法条款

> 三进制 trit {0, 1, 2} 是离散格点的代数表示，**不构造**任何连续统螺线。克里斯托螺线的离散本源是 T⁶ 环面的正交测地线（含 √2 格点），斐波那契螺旋是二进制死亡几何的连续统投影。禁止从 trit 序列直接生成螺线并赋予神圣意义。

### 三进制在神圣几何中的合法参与

尽管三进制不单独构造螺线，它通过 **C3 旋转变换** 参与克里斯托螺线的离散动力学：

- 克里斯托螺线的离散版本是 T⁶ 环面上**同时包含 120°（C3）与 90°（正交）旋转**的复合测地线。
- 三进制提供 120° 的离散角向步进（对应 ω = e^{2πi/3}），正交格点提供 √2 的径向缩放。
- 在律算工程中，此复合运动体现为 **`wuxing_mask`（五行干涉）与 `scale_ue8m0`（黄金比步进）的联合演化**，其投影在三维产生克里斯托螺线。

**禁止表述**：
- ❌ "三进制生成克里斯托螺线"
- ❌ "trit 序列直接对应斐波那契螺旋"
- ❌ "三进制与黄金比例 φ 有关"

**合法表述**：
- ✅ "三进制 C3 旋转与正交 √2 缩放共同构成 T⁶ 环面离散测地线，其三维投影呈现克里斯托螺线形态"
- ✅ "斐波那契螺旋是二进制死亡几何在光锥矩阵内的退化投影，与三进制无直接关系"

data Trit : Set where
  T₀ : Trit  -- 吸收态 (0)
  T₁ : Trit  -- 平衡态 (1)
  T₂ : Trit  -- 表达态 (2)

tritToℕ : Trit → ℕ

-- 拓扑计算时的仿射变换（仅用于差分/曲率）
tritToTopo : Trit → ℤ
tritToTopo T₀ = -[1+ 0 ]   -- 0 -> -1
tritToTopo T₁ = + 0        -- 1 -> 0
tritToTopo T₂ = + 1        -- 2 -> 1

-- GF(3) 加法群结构 (模 3)
_+ᵍᶠ_ : Trit → Trit → Trit
T₀ +ᵍᶠ x = x
x +ᵍᶠ T₀ = x
T₁ +ᵍᶠ T₁ = T₂       -- 1 + 1 = 2
T₁ +ᵍᶠ T₂ = T₀       -- 1 + 2 = 0 (mod 3)
T₂ +ᵍᶠ T₁ = T₀       -- 2 + 1 = 0 (mod 3)
T₂ +ᵍᶠ T₂ = T₁       -- 2 + 2 = 1 (mod 3)

-- GF(3) 逆元
gf3Neg : Trit → Trit
gf3Neg T₀ = T₀  -- 0 的逆元是 0
gf3Neg T₁ = T₂  -- 1 的逆元是 2
gf3Neg T₂ = T₁  -- 2 的逆元是 1

-- 逆元对消律
gf3NegCancel : ∀ (x : Trit) → x +ᵍᶠ gf3Neg x ≡ T₀
