## 2. 关键认识论澄清(避免重走弯路)

4. **PR #8611 work ≠ 大衍理论被验证**。让 fix work 的是望远镜算术(`nTarget = nOld + nctel - 1` + HDU thunk 不递归 + `raiseS 1`)。CRT 是**架构隐喻**(decompose-into-index/field-dims 的精神,legitimate),但代码不算 CRT 余数;torus/幻方/声子/相位对齐/极限环在代码和 PR 描述里**都没出现**。「fix work → 大衍被证实」是 non-sequitur。
