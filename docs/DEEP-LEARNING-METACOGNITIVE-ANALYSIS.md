### 2.2 范畴分离原则

这是项目最核心的方法论创新：

| 范畴 | 用途 | 禁止操作 | 示例 |
|------|------|---------|------|
| **根数学** | GF(3) Trit, 不变量 | 浮点近似 | `Trit = {-1, 0, +1}` |
| **结构学** | Tryte (729态), 纤维丛截面 | 直接 I/O | `Tryte = Vec Trit 6` |
| **耦合域** | PackedTryte5 (243态), 地址翻译 | 参与主权运算 | `pack5 : Vec Trit 5 → ℕ` |
| **密度** | 文明层级投影 | 跨层级混用 | `12d, 24d, 144d` |

**违宪行为示例**：
```agda
-- ❌ 错误：将 PackedTryte5 直接用于主权运算
let packed = pack5 trits in
evolveState packed  -- 类型错误！

-- ✅ 正确：先解包为 SovereignSection
let section = unpackQsToSection qs in
evolveState section  -- 类型正确
```

**深层原因**：
不同范畴对应不同的抽象层级和文明密度。混淆范畴会导致：
1. **信息丢失**：从 729 态压缩到 243 态是不可逆的
2. **语义污染**：工程编码混入逻辑运算
3. **维度坍缩**：高维拓扑被低维投影覆盖

## 四、技术实现层面：代码的深度解析

### 4.1 GF(3) 三进制的精妙设计

#### Trit 数据类型

```agda
data Trit : Set where
  T- : Trit  -- -1 (吸收态/阴)
  T0 : Trit  --  0 (平衡态/中)
  T+ : Trit  -- +1 (表达态/阳)
```

**为什么选择 {-1, 0, +1} 而非 {0, 1, 2}？**

1. **对称性**：正负平衡，便于表示手性（左旋/平衡/右旋）
2. **对消灭语义**：T+ ⊕ T- = T0（1 + (-1) = 0）直观表达虚实对消灭
3. **易经映射**：
   - T- = 阴爻 ⚋
   - T0 = 中道 ☯
   - T+ = 阳爻 ⚊

#### 加法表（模 3）

```
⊕  | T-  T0  T+
---+------------
T- | T+  T-  T0
T0 | T-  T0  T+
T+ | T0  T+  T-
```

**关键性质**：
- T+ ⊕ T- = T0（对消灭）
- T+ ⊕ T+ = T-（1 + 1 = 2 ≡ -1 mod 3）
- T- ⊕ T- = T+（-1 + -1 = -2 ≡ 1 mod 3）

**物理意义**：
- 两个阳相遇产生阴（物极必反）
- 两个阴相遇产生阳（否极泰来）
- 阴阳相遇归于平衡（中和）

### 4.2 LCM 模运算的数学基础

#### 主权 LCM 的计算

```
SOVEREIGN_LCM = 3^11 × 2^16
              = 177,147 × 65,536
              = 11,609,505,792
```

**为什么是这个数？**

1. **3^11**：来自仲吕闭合的乘法因子
   - 十二律损益链涉及 3 的幂次
   - 3^11 = 177,147 是黄钟余数

2. **2^16**：来自工程实现的位移操作
   - `>> 16` 等价于除以 2^16
   - 对齐 16 位定点数格式

3. **乘积**：极向与环向同步归零的工程周期
   - 当累加器达到此值时，极向相位和环向相位同时归零
   - 完成一个完整的"主权呼吸"周期

#### 仲吕闭合操作

```python
def zhonglv_closure(acc: int) -> int:
    """acc ↦ (acc * 3^11) >> 16"""
    return (acc * 177147) // 65536
```

**数值验证**：
```
zhonglv_closure(65536) = (65536 * 177147) // 65536
                       = 177147  # 仲吕余数 → 黄钟余数
```

**几何意义**：
- 极向缠绕从模 12 展开为模 144
- 环向缠绕从模 10 升维为模 46
- 主权状态机虚实比归零
- T⁶ 环面的和乐归零
